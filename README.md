# Snake Game

A classic Snake game built with vanilla JavaScript and HTML Canvas — no libraries, no frameworks.

## Play
[Click here to play](https://yugandharvasudev.github.io/SnakeGame)

## Controls
| Key | Action |
|-----|--------|
| R | Start / Restart |
| Arrow Keys | Move snake |
| P | Pause / Resume |
| E | Easy mode (wraparound) |
| H | Hard mode (walls) |
| Esc | Main menu |

## Features
- Dual game modes — wraparound and walled
- Dynamic speed scaling as score increases
- Self-collision and wall collision detection
- Input buffering to prevent illegal direction reversals
- Pause and main menu navigation

## How to run locally
Open `index.html` in any browser — no setup needed.
